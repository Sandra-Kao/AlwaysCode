用 HTML 和 CSS 製作 select Box 的 Pleceholder

<input>和<textarea>HTML元素都天生自帶 placeholder 功能，唯獨 <select> 下拉選單在UI上想設定類placeholder功能會相對麻煩。

參考資料： HTML | placeholder Attribute - GeeksforGeeks

Plaseholder (佔位符)

Plaseholder 屬性主要被建立在可以和使用者互動的 HTML element 元素上，用以在填入 content 之前，提示使用者可以接著執行的操作，並且 plaseholder 提示文字不影響該元素包含的 content 。

範例：


        <element plaseholder="">
        <!-- element 元素  /  plaseholder 一種屬性-->        
        <!-- 可以使用 plaseholder 屬性的 HTML 元素有 -->
        <!-- <input> 和 <textarea> -->

範例：




        <input type="text" name="fname" 
                         placeholder="姓名"> 
        <br> 
        <input type="text" name="lname" 
                          placeholder="電話"> 
        <br> 
        <input type="submit" value="送出" disabled>

範例：


        <textarea rows="3" cols="40" 
            placeholder="這是一個可以填多行文字的區塊"> 
        </textarea>

<select> Box placeholder

在 <select> Box 中的 placeholder，其實是一個 <option>。

利用控制第一個 <option> 的樣式，來達成 (類)placeholder 的效果。


Select 範例01：

用 disabled 跟 selected 就能達到 placeholder 效果。


        <select required>
            <option  disabled selected>Choose a drink</option>
            <option>Coffee</option>
            <option>Tea</option>
            <option>Milk</option>
        </select>

Select 範例02：

用 disabled, selected 再加上 HTML5 的 hidden 屬性製作選單的 placeholder 不會出現在下拉選單的效果。


        <select required>
            <option disabled selected hidden>Choose a drink</option>
            <option>Coffee</option>
            <option>Tea</option>
            <option>Milk</option>
        </select>

Select 範例03：

在 css 中設定 select:required:invalid。Required 屬性的 type 是 true or false (boolean)，表示表單為必填值；Invalid 設定會根據元素來驗證元素的值無效，在此表示 placeholder 值無效。

在 HTML 中設定 select required 表示必填；在 類placeholder 的 option 上設定 disabled 表示不可被選擇； selected 則表示此為 select 中的預設 option 。


        <style>
            select:required:invalid {
                color: #666;
            }
            
            option {
                color: #000;
            }
        </style>
                                    

        <select required>
            <option disabled selected hidden>Choose a drink</option>
            <option>Coffee</option>
            <option>Tea</option>
            <option>Milk</option>
        </select>
