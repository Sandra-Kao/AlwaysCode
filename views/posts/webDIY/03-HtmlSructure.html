HTML Document Structure

接續著上篇 設計一份HTML ，我們接著來談我配置的HTML docutment structure是怎麼配置的？以及為什麼這樣配置？

HTML的組成

HTML的Tag(標籤)通常由帶有小於(<)和大於(>)的開頭和結尾標籤組成， 且結尾標籤必須帶有 / (斜線符號)，例如：


        <h3> ... </h3> <!-- h3表示3號字級標題 -->
        <div> ... </div> <!-- div表示區塊 -->

但也有少數例外。有些Tag能自己單獨結尾，並且不一定要帶有 / (斜線符號)，例如：


        <hr>
        <hr /> <!-- hr表示水平線 -->

        <br>
        <br /> <!-- br表示斷行 -->

HTML Tag 可以帶有不同的屬性和數值，例如：

                                    
        <a id="myLink"
              href="#HtmlStructureTags"
              class="my-link"
              style="color: #ff8888;"
              target="_blank">
        Html Document Structure
        </a>

        <!-- <a> 表示超連結，標籤內文字為網頁上顯示的連結文字 -->
        <!-- id 表示指定HTML文件中的元素，id必須是唯一值 -->
        <!-- href 表示連結目標位置 -->
        <!-- class 表示自定義的多個樣式style名稱，可重複使用在不同的標籤上 -->
        <!-- style 表示設定的標籤樣式 -->
        <!-- target="_blank" 表示連結需另開網頁新視窗 -->
